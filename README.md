## 프로젝트 개요
K-리그 경기 분석 프로젝트는 머신 러닝 알고리즘을 활용하여 K-리그 경기 데이터를 분석하고 결과를 예측합니다. 데이터 전처리, 특징 선택, 다양한 분류 모델 학습 등을 포함하고 있습니다.

- Python 3.x
- pandas
- numpy
- matplotlib
